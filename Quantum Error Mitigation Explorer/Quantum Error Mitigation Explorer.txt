Zero Noise Extrapolation (ZNE) - The Workhorse:
1. Run circuit at noise level 1x, 2x, 3x, ... (amplify noise)
2. Measure success probability P(λ) for each
3. Fit exponential: P(λ) = A - B·exp(-C·λ)
4. Extrapolate back to λ=0 (zero noise)
5. Result: Ideal outcome with only 2-3x quantum overhead
Measurement Error Mitigation (MEM):
1. Calibrate: Measure prepared |0⟩ and |1⟩ states
2. Build error matrix M: P(measured|prepared)
3. Invert: p_true = M⁻¹ · p_measured
4. Corrects readout errors (~0.5-2% per qubit)

What Errors Look Like:
•	Gate errors: ~0.1-1% per gate (single vs two-qubit)
•	Decoherence: T2 ~20-50 μs (1 μs per gate → ~2% per gate)
•	Measurement: ~0.5-2% error rate per qubit
•	Crosstalk: Neighboring qubits interfere
•	After 100 gates: ~10-50% total error
